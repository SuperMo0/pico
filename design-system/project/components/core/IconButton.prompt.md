Icon-only button for header actions (search, cart, account) and quantity steppers.

```jsx
<IconButton icon={<i data-lucide="shopping-bag" />} label="السلّة" badge={2} />
<IconButton icon={<i data-lucide="search" />} label="بحث" variant="outline" />
```

Variants `ghost | outline | solid`. `round` for circular. `badge` shows a count.
